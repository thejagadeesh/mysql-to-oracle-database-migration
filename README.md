# MySQL to Oracle Database Migration

This project demonstrates the migration of a relational database from MySQL to Oracle using Oracle SQL Developer Migration Workbench.

---

## 🚀 Overview

- Performed end-to-end database migration from MySQL to Oracle  
- Converted database objects including tables, constraints, relationships, and indexes  
- Handled migration challenges such as repository issues, account locks, and constraint violations  
- Validated data integrity using SQL queries and row count checks
